Support Request Guidance

We want to make sure we can get your issue resolved as quickly as possible.
If we need as much detail about the problem in your first ticket then this will speed the time to resolution.
Please have in mind the following when wriitng your ticket.

  • What program were you running or operation were you performing when the issue came about?
  • Is the issue reproducible? What are the exact steps which cause the issue (Steps to Reproduce)?
  • Would a picture better describe the issue?
